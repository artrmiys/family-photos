# 📷 Фото для фамильного древа

Фотографии для сайта https://family.185.81.165.121.sslip.io — подгружаются прямо из этого репозитория, место на сервере не тратится.

## Как добавить фото человека

1. Положи фото в папку [`photos/`](photos/) — можно прямо через веб-интерфейс GitHub: **Add file → Upload files**
2. Называй файл латиницей, просто и понятно: `ivan-petrov.jpg`, `baba-masha.jpg`
3. На сайте (в режиме редактора) открой человека → **✎ Редактировать** → в поле «Фото» впиши имя файла: `ivan-petrov.jpg`

Готово — фото появится в кружке на графе и в карточке.

## Рекомендации

- Формат: **JPG**, лучше квадратные (лицо по центру) — на сайте фото показывается в круге
- Размер: достаточно 400–800 px по стороне, большие файлы дольше грузятся у посетителей
- ⚠️ Репозиторий **публичный** (иначе прямые ссылки не работают) — не выкладывай фото, которые нельзя показывать всем

## Как это работает

Сайт берёт файлы по прямой ссылке:
`https://raw.githubusercontent.com/artrmiys/family-photos/main/photos/<имя-файла>`

Базовая ссылка задана в настройках сайта (кнопка 🖼 в режиме редактора), поэтому в карточке человека достаточно только имени файла. Полные ссылки на любые другие картинки в интернете тоже работают.
